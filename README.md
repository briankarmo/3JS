## 3D Portfolio

This is a custom 3D developer portfolio built with React, Three.js, and Tailwind CSS. It features an interactive 3D hero, a projects showcase, a technologies grid, testimonials, and a contact section. The content is tailored for a noâ€‘contact presentation (no personal last names or external profile links).

### Tech Stack
- React, Vite
- Three.js, @react-three/fiber, @react-three/drei
- Tailwind CSS, Framer Motion

### Run Locally
1. Install dependencies:
```
npm install
```
2. Start the dev server:
```
npm run dev
```
3. Open the development server in your browser (default port 5173).
